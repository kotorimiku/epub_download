# 构建

#### 前提

- [Rust](https://www.rust-lang.org/tools/install)
- [Node](https://nodejs.org/en)
- [pnpm](https://pnpm.io/installation)

#### 步骤

#### 1. 克隆本仓库

```
git clone https://github.com/kotorimiku/epub_download
```

#### 2.安装依赖

```
cd epub_download
pnpm install
```

#### 3.构建(build)

```
pnpm tauri build
```
